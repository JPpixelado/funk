# Guia de Contribuição — sexta à noite do funk

Seja bem-vindo(a) ao projeto **sexta à noite do funk**! Este jogo, feito em Python com pygame, conta uma história "absolute cinema" e sua colaboração é super valiosa para melhorarmos ainda mais.

## Como contribuir

1. **Fork este repositório**  
   Clique em "Fork" na página principal do repositório para criar uma cópia na sua conta.

2. **Clone o repositório do seu fork**
   ```
   git clone https://github.com/seu-usuario/funk.git
   ```

3. **Crie uma branch para sua alteração**
   ```
   git checkout -b minha-contribuicao
   ```

4. **Implemente sua melhoria ou correção**
   - Siga a [PEP 8](https://pep8.org/) para o código Python.
   - Se for um novo recurso, adicione testes se possível.
   - Para bugs, tente adicionar um teste que reproduza o erro corrigido.

5. **Faça commit das alterações**
   ```
   git add .
   git commit -m "Descrição clara da alteração"
   ```

6. **Envie sua branch para o GitHub**
   ```
   git push origin minha-contribuicao
   ```

7. **Abra um Pull Request (PR)**
   - Detalhe o que sua contribuição faz e por quê.
   - Relacione issues relevantes, se houver.

## Boas práticas

- Sempre escreva mensagens de commit claras e objetivas.
- Faça pequenas alterações por Pull Request. Evite grandes PRs com múltiplos objetivos.
- Revise e teste suas alterações antes de enviar.
- Seja respeitoso e aberto ao feedback nos Pull Requests.

## Dúvidas ou sugestões?

Abra uma [issue](https://github.com/JPpixelado/funk/issues) e descreva seu problema ou ideia. Ficaremos felizes em ajudar!
